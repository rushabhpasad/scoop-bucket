# scoop-bucket

Scoop bucket for [MD Editor](https://github.com/rushabhpasad/md-editor) — a lightweight, cross-platform desktop Markdown editor.

## Install

```powershell
scoop bucket add rushabhpasad https://github.com/rushabhpasad/scoop-bucket
scoop install md-editor
```

## Update

```powershell
scoop update md-editor
```

## Uninstall

```powershell
scoop uninstall md-editor
```

## Contents

| App | Description | Version |
|---|---|---|
| `md-editor` | Lightweight cross-platform desktop Markdown editor | latest |

## Manifest updates

Manifests in this bucket are updated automatically by a GitHub Actions workflow in the main [md-editor](https://github.com/rushabhpasad/md-editor) repository whenever a new release is published.
